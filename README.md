# NURA WEAR - Estilo tipo Zara

Web rediseñada con una estética muy limpia tipo gran marca de moda:
- Cabecera fija minimalista
- Logo grande centrado
- Menú lateral
- Portadas a pantalla completa
- Categorías Mujer / Hombre
- Tienda en cuadrícula limpia
- Carrito lateral
- Todo en español

No es una copia exacta de Zara: mantiene identidad propia de NURA WEAR y Menorca.
